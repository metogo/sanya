# Banner 图片目录

## 📁 放置你的 Banner 图片到这里

### 默认 Banner 图片

将你的 banner 图片命名为以下文件名之一：

- `sanya-hero.jpg`
- `sanya-hero.png`
- `sanya-hero.webp`

### 图片规格要求

#### 推荐尺寸

- **宽度**: 1920px - 3840px
- **高度**: 600px - 1200px
- **比例**: 16:5 或 3:1

#### 文件大小

- **原图**: 可以 2-5MB（会自动压缩）
- **优化后**: 系统自动压缩到 50-200KB

#### 支持格式

- ✅ JPG/JPEG
- ✅ PNG
- ✅ WebP
- ❌ GIF (不支持动图优化)

### 图片内容建议

1. **三亚海滩风景** - 蓝天、白云、碧海
2. **日落/日出景观** - 天空渐变色
3. **热带风情** - 椰树、沙滩
4. **标志性建筑** - 凤凰岛、南山寺
5. **高角度俯瞰** - 展现全景

### 重要提示

⚠️ **文字区域**: Banner 中心会覆盖文字，确保重要景色不在正中央

✅ **高对比度**: 选择明亮或对比强烈的图片，让白色文字清晰可见

🎨 **色彩建议**: 蓝色系（海洋）、橙色系（日落）效果最佳

### 示例图片来源

免费高质量图片网站：

- Unsplash: https://unsplash.com/s/photos/sanya
- Pexels: https://pexels.com/search/tropical-beach
- Pixabay: https://pixabay.com/

### 快速开始

1. 下载你喜欢的三亚风景图片
2. 重命名为 `sanya-hero.jpg`
3. 放到这个目录 (`public/images/banners/`)
4. 完成！图片会自动通过 Vercel CDN 加速

### 查看效果

```bash
npm run dev
# 访问 http://localhost:3000
# Banner会显示在页面顶部
```

### 多语言文字

Banner 会根据用户选择的语言显示不同文字：

- 🇨🇳 中文: "探索三亚之美"
- 🇬🇧 英文: "Explore the Beauty of Sanya"
- 🇷🇺 俄文: "Исследуйте красоту Санья"
